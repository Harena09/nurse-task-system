# Nurse Task Management System

## Overview
The Nurse Task Management System is a software application designed to help nurses and healthcare staff organise, manage, and track daily tasks efficiently within a healthcare environment.

This project was developed as part of my Computer Science studies to improve task coordination, reduce missed activities, and support better workflow management.

---

## Features
- User authentication and login
- Task creation and assignment
- Task status tracking
- Priority management
- Dashboard for viewing tasks
- Responsive user interface
- Database integration for storing task information
- Handover System
- Mood tracker for nurses
- AI nurse Assistant

---

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
